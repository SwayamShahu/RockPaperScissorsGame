## 🧠 How This Rock Paper Scissors Game Works

1. User selects Rock, Paper, or Scissors
2. Computer generates a random choice using:

   ```javascript
   Math.random()
   ```
3. The result is calculated using `if-else` conditions
4. Output is displayed dynamically on the screen

### Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
